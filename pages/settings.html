GitHub Pages Deployment

To run this app from GitHub Pages:

1. Create a new GitHub repo (public)
2. Upload all files maintaining the folder structure
3. Go to repo Settings → Pages → Source: main
4. Your app will be at username.github.io/repo-name
5. Add your API key in Settings after opening the live URL
